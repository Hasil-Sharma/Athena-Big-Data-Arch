# Athena

Specifications:
- Java 7
- Scala 2.11.12
- sbt 0.12.16

Running Sample Storm Project:

`sbt clean compile assembly`

`java -jar target/scala-2.11/athena-assembly-0.1-SNAPSHOT.jar`
